Temple University Sustainability Map (updated)

ENST 4198

Map by Megan Platt, Caroline Muehlbronner, and Max Schmader

Capstone Group Project

This interactive map allows users to explore sustainability intiatives at Temple University's Main Campus. Each marker on the map represents one of these intitiatives. If users click on the marker, a popup will appear stating the name of the place as well as a small description explaining how the place is active in Temple's sustainability initiative.
